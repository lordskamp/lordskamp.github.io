# Рівні гри «Шифр»

Кожна категорія має один однойменний JSON-файл. Щоб додати рівень, додайте один рядок:

```json
{ "text": "Текст рівня", "source": "Автор або джерело | https://посилання.example" }
```

Це всі дозволені поля. ID, номер рівня, доступ і складність формуються автоматично. Посилання після ` | ` необов’язкове, але джерело перед ним — обов’язкове.

Публічний репозиторій може містити лише безкоштовні рівні та явно визначені preview-рівні. Платні файли з такою самою простою структурою тримайте поза репозиторієм і завантажуйте до Cloudflare KV командою з `docs/SHYFR.md`.
